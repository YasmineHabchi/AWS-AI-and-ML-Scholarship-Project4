Project: Build an ML Workflow For Scones Unlimited On Amazon SageMaker

Overview: This project is a part of the Assessment in Udacity's AWS Machine Learning Engineer Nanodegree Program.

project files: 

        1/ Starter.ipynb represents a complete machine-learning workflow for image classification. This includes preprocessing the Scones Unlimited image dataset, model training, deployment, and monitoring using Amazon SageMaker and stepfunction.
        2/StepFunction.json: This file has the step function code.
        3/lambda.py: this file encompasses the 3 lambda functions that I used in the StepFunction
        4/StepFunction.png: is the screenshot of the StepFunction
